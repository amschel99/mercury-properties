# Shareable Links for Mercury Properties

Use these links in your social media posts, WhatsApp messages, emails, etc. When clicked, they will automatically open the appropriate application form.

## For House Seekers (Renters)

**Full URL:**
```
https://yourdomain.com/apply/renter
```

**Short version (if you have a domain):**
```
yourdomain.com/apply/renter
```

**What happens:** Opens the "Find Your Home" application form modal instantly.

## How It Works

1. User clicks the link
2. They're redirected to the homepage
3. The appropriate modal opens automatically
4. They fill out the form
5. Form submits and they see a success message

The URL automatically cleans up after opening the modal (no query params visible).
